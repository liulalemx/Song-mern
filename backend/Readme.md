<h1 align="center">Song-Mern API</h1>

## Techstack

- [Typescript](https://www.typescriptlang.org/) - language
- [Express](https://expressjs.com/) - web framework
- [MongoDB](https://www.mongodb.com/) - database
- [Zod](https://zod.dev/) - data validation
- [Docker](https://www.docker.com/) - packaging
- [Render](https://render.com/) - deployment

## Endpoints

For more detailed documentation, check the [api docs](https://documenter.getpostman.com/view/21591099/2sA2r824XA).

### `/v1/songs/`

- Method: `POST`

### `/v1/songs/`

- Method: `GET`

### `/v1/songs/:id`

- Method: `GET`

### `/v1/songs/:id`

- Method: `PUT`

### `/v1/songs/:id`

- Method: `DELETE`

### `/v1/stats/`

- Method: `GET`

### `/v1/stats/albums`

- Method: `GET`

### `/v1/stats/artists`

- Method: `GET`

### `/v1/stats/genres`

- Method: `GET`

## How to run locally

### Prerequisites

- [nodejs](https://nodejs.org/en/)

1. Clone the repository
1. Run `npm install`
1. Run `npm run build` on the root directory
1. Run `npm run dev`
